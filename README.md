## 🚀 **QUICK START - FUNCIONA EM 5 MINUTOS**

### 📋 **Pré-requisitos**
```bash
✅ Python 3.8+
✅ pip ou conda  
✅ Chave OpenAI API (gratuita)
```

### ⚡ **Instalação Express**
```bash
# 1️⃣ Clone o repositório
git clone https://github.com/danielgomesresende/ia-analista-negocios.git
cd ia-analista-negocios

# 2️⃣ Instale dependências (30 segundos)
pip install -r requirements.txt

# 3️⃣ Configure variáveis (opcional)
cp .env.example .env
# Adicione sua chave OpenAI

# 4️⃣ Execute demo (30 segundos)  
streamlit run app.py

# 🎉 Abra http://localhost:8501
```

### 🔧 **Configuração Personalizada**
